What is Social Login?

King Social Login is a powerful authentication plugin for king media that allows users to sign in and register using their social media accounts such as Google and Facebook.

It simplifies the login process by eliminating traditional registration forms, reduces signup friction, and improves user experience by providing fast, secure, and convenient access to your community.

Key Features

  • Google & Facebook Login
    Allow users to sign in or register instantly using their Google or Facebook accounts.
  • One-Click Authentication
    Users can access your website without creating a traditional account or remembering passwords.
  • Automatic Account Creation
    New users are automatically registered on first login with their social profile data.
  • Secure OAuth Integration
    Uses official Google and Facebook authentication APIs for safe and secure login.
  • Modern & Responsive UI
    Clean login buttons designed to match your Q2A / King CMS theme and mobile devices.
  • Fast Setup
    Simple configuration using API keys and redirect URLs — no complex setup required.
  • No Core File Modifications
    Installs as a plugin without editing Q2A or King CMS core files.

Requirements

  • King CMS version: 1.8+ recommended
  • HTTPS (SSL Certificate): Required for Google and Facebook authentication
  • Google OAuth API credentials: Client ID and Client Secret from Google Cloud Console
  • Facebook App credentials: App ID and App Secret from Facebook Developers
  • Valid redirect URLs: Must match exactly in provider settings (Google & Facebook)
  • PHP cURL enabled: Required for secure communication with authentication providers

Why This Plugin?

Most king media websites rely on traditional registration forms, which can slow down user onboarding and reduce sign-ups.

This plugin enables fast and secure social login using Google and Facebook, allowing users to register and log in instantly — improving user experience, increasing engagement, and simplifying account access.
