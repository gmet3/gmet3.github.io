What is King SEO Booster?

King SEO Booster is a powerful SEO optimization plugin designed for King Media Bundle – Viral Magazine Script. It helps you improve Google visibility, enhance indexing, and manage metadata easily — without editing core files.

Control SEO titles/descriptions, boost social sharing previews, and keep your site lightweight and fast.

Key Features

  • Advanced SEO Optimization
    Improve rankings and visibility with structured metadata and SEO-friendly output for King Media pages.
  • Automatic Meta Tags
    Manage SEO titles, descriptions and indexing rules for posts, pages, categories and tags.
  • Open Graph & Twitter Cards
    Better previews when sharing your content on Facebook, Twitter and messaging apps.
  • Indexing & Crawl Improvements
    Helps search engines discover content faster and improves overall crawl efficiency.
  • No Core Hacks Required
    Integrates cleanly via the plugin layer — no need to modify core files or themes.
  • Lightweight & Fast
    Optimized codebase to keep your King Media site performant and responsive.

Requirements

  • Script: King Media Bundle – Viral Magazine Script
  • PHP: 7.4+ recommended
  • Permissions: Ability to upload/install plugins on your server

Installation

Upload the plugin folder to your king-plugin/ directory, then enable it from the Admin Panel. Clear cache if your script uses caching.
