What is Gmet3?

Gmet3 is a responsive, elegant and fast gallery system built for painters, craftsmen, and creative professionals. It allows you to showcase your work, receive client selections, and manage content with a clean admin interface.

Carefully Designed

Every detail is hand-crafted: modern typography, smooth animations, and a professional golden accent tone. Fully optimized for both mobile and desktop screens.

Features

  • ✔ Admin dashboard to manage posts, users, and comments
  • ✔ AJAX likes & category filters
  • ✔ Notifications when clients select images
  • ✔ Support for YouTube, Vimeo, and local videos
  • ✔ Fully responsive — built for phones first
